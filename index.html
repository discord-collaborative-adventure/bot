Bienvenue sur notre serveur Discord

DisCa est un serveur discord qui vous permettra de réaliser vos projets, vous nous les présentez et on vous aidera. Vous pouvez aussi venir nous faire un petit coucou ;)

Voici les informations sur le bot

Règlement et conditions d'utilisation

En invitant le bot sur votre serveur vous consentez à ce que l'ensemble des messages envoyés sur celui-ci soient enregistrés par le bot

Vous avez également l'obligation de prévenir les membres dès leur arrivée que leurs messages seront enregistrés

Si un membre de votre serveur a une réclamation à faire vous en serez seul résponsablle

Si vous êtes le propriétaire du serveur vous pouvez à tout moment demander les logs de messages en envoyant un message privé à tv.fr_ sur discord

L'ensemble des utilisateurs doivent se soumettre aux règles suivantes :

Ne pas enfreindre les TOS de discord

Utiliser le bot uniquement dans son usage prévu

En aucun cas lier le bot avec quelques activitées controversées que ce soit

Conditions d'utilisation :

Vous devez être âgé d'au moins 15 ans pour utiliser le bot seul

Si vous avez moins de 15 ans vous devrez être accompagné par un représentant légal pour l'utiliser

Si vous êtes une personnalité publique vous vous engagez à ne pas faire de la promotion de quoi que ce soit en faisant un lien avec le bot

Si vous êtes connu(e) pour des abus avec le bot vous pourrez avoir l'interdiction temporaire ou définitive de l'utiliser

Stockage de données :

Par mesure de sécurité tous les messages sont stockés et conservés sur un fichier. Vos données ne sont pas revendues ni divulguées à des entreprises ou particuliers.

Le seul cas où vos messages pourraient êtres divulgués : En cas de problème grave nécessitant un accès aux logs (accès réservé aux owner)

Situations nécessitant les logs : Menaces, Harcèlement, Piratage, Pornographie, Nudité

Dans tous les cas tv.fr_ a accès 24h/7j aux logs et si la situation l'exige vous serez sanctionné

Devoirs :

• Vous devez respecter entièrement le présent règlement

Introduction :

@tv.fr_ ainsi qu'une minorité du staff ont un accès permanent aux logs des messages envoyés sur ce serveur. Nous faisons ceci dans la limite des TOS de discord, ces données peuvent êtres exploitées de diverses façons.

Exploitation des données :

Les données suivantes sont stockées :

• Nom d'utilisateur

• ID unique d'utilisateur

• Contenu du message

• Si votre compte est un bot ou non

• La date et l'heure du message

• Le lien d'accès au message

• L'ID unique du message

Comment sont-elles exploitées :

Elles sont exploitées des façons suivantes :

• Surveillance du chat afin de garantir la sécurité des membres

• Stockage des messages afin de garantir des preuves tangibles et immuables en cas de conflit

• Formations du Staff (exemples, démonstrations)

• Promotion du serveur sur les réseaux sociaux ( votre nom ne sera pas mentionné )

• Elles ne seront jamais vendues ou échangées avec d'autres serveurs / entreprises / organisations /associations

Comment demander la suppression de mes données :

Vous pouvez demander à n'importe quel instant la suppression de vos données dans les cas suivants :

• Vous quittez le serveur et ne souhaitez pas laisser de traces

• Vous êtes mal à l'aise avec l'enregistrement et le stockage de vos données

• Si vous êtes mineur(e) et qu'un de vos représentants légaux avec preuve sur l'honneur demande la suppression de vos données

Les demandes de suppression pourront être refusées dans les cas suivants :

• Vous demandez la suppression de données d'une autre personne

• Vous faites l'objet d'une enquête par notre Staff

• Vous êtes classés comme utilisateur à risques

Comment nous enregistrons vos données :
